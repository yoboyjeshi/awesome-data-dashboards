## BI & Visualization Tools

| Tool | Type | Best For |
|------|------|----------|
| [Amazon QuickSight](https://aws.amazon.com/quicksight/) | Cloud BI | AWS-integrated dashboards, embedded analytics |
| [Tableau](https://www.tableau.com/) | Enterprise BI | Interactive visualizations, drag-and-drop |
| [Power BI](https://powerbi.microsoft.com/) | Enterprise BI | Microsoft ecosystem, self-service analytics |
| [Looker](https://cloud.google.com/looker) | Cloud BI | Data modeling with LookML, Google Cloud |
| [Metabase](https://www.metabase.com/) | Open Source BI | Simple setup, no-code queries |
| [Apache Superset](https://superset.apache.org/) | Open Source BI | Enterprise-grade, Python-based |
| [Redash](https://redash.io/) | Open Source BI | SQL-first, query-focused dashboards |

## Open Source Frameworks

- [Streamlit](https://streamlit.io/) — Build data apps in pure Python, zero frontend needed
- [Dash (Plotly)](https://dash.plotly.com/) — Python framework for analytical web applications
- [Grafana](https://grafana.com/) — Monitoring and observability dashboards
- [Observable](https://observablehq.com/) — JavaScript notebooks for data visualization
- [Evidence](https://evidence.dev/) — SQL-based reporting with Markdown
- [Tremor](https://www.tremor.so/) — React components for building dashboards
- [Chartbrew](https://chartbrew.com/) — Open-source platform connecting to databases and APIs
